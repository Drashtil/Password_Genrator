# Password Generator

A simple Python Password Generator that creates strong and random passwords based on user preferences.

## Features

* Generate random passwords
* Custom password length
* Includes letters, numbers, and symbols
* Fast and easy to use
* Beginner-friendly project

## Technologies Used

* Python
* Random Module
* String Module

## How It Works

The program:

1. Takes password length from the user
2. Uses random characters to generate a secure password
3. Displays the generated password instantly
